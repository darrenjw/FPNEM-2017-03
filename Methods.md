# Commonly used Spark methods

## Transformations of an `RDD[T]`

* distinct
* filter
* flatMap
* map
* persist
* sample
* zip

## Actions on an `RDD[T]`

* aggregate
* collect
* count
* fold
* foreach
* reduce
* take

## Transformations of a Pair RDD, `RDD[(K,V)]`

* aggregateByKey
* groupByKey
* keys
* mapValues
* reduceByKey
* values

## Actions on a `RDD[(K,V)]`

* countByKey

## `DataFrames`

* toDF
* col
* drop
* select
* withColumn
